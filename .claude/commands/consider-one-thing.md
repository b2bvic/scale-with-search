# The ONE Thing (Keller)

## When to Use

Invoke when scattered across too many priorities, when progress is wide but shallow, when you need to cut through noise and identify the single action that matters most right now, or when everything seems equally important (which means nothing has been prioritized).

## Execution

1. **Define the timeframe.** Are you asking about today, this week, this quarter, or this year? The ONE Thing shifts depending on the horizon.

2. **List all active priorities.** Everything you're working on, thinking about, or feel you should be doing. Include the ones you're avoiding — those are often the most important.

3. **Ask the focusing question.** For each item, ask: "What's the ONE thing I can do, such that by doing it, everything else becomes easier or unnecessary?"

   This question has a specific structure:
   - "What's the ONE thing" — forces singular focus
   - "I can do" — must be actionable, not aspirational
   - "Such that by doing it" — demands causal leverage
   - "Everything else becomes easier or unnecessary" — the litmus test for true priority

4. **Identify the lead domino.** Among all candidates, which one, if toppled, would set the most other dominoes in motion? This is not about what's most urgent or most visible. It's about what's most *causal*.

5. **Stress-test the selection.** Ask:
   - If I could only do this one thing today/this week, would I feel the time was well spent?
   - Does this create capacity or just consume it?
   - Am I choosing this because it's important, or because it's comfortable?

6. **Name the commitment.** State the ONE Thing as a concrete action with a clear completion state. Not "work on the product" but "ship the landing page by Thursday."

## Output Format

```
## The ONE Thing: [Timeframe]

### All Active Priorities
1. [Priority]
2. ...

### Focusing Question Applied
| Priority | "Everything else easier or unnecessary?" | Causal Leverage |
|---|---|---|
| [Priority] | [Yes/No — why] | [High/Med/Low] |

### The Lead Domino
[The single action identified]

### Why This One
[How completing this makes other priorities easier or unnecessary]

### Commitment
[Concrete action + completion criteria + deadline]
```
